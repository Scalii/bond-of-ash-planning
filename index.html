Inventory System

+

Manage items, equipment and resources. Support stacking, slots, crafting inputs and replication.

  • Define item data structures and custom GameInstance array
  • Implement pickup and use logic via base classes
  • Design and implement inventory UI (grid/list & quick bar)
  • Add network replication for inventory
  • Integrate saving/loading of inventory data

Ability System & Skill Tree

+

Implement abilities, attributes and effects using GAS. Provide a skill tree for progression.

  • Enable Gameplay Ability System plugin and configure dependencies
  • Design Attribute Sets (health, stamina, dragon bond, etc.)
  • Create base Gameplay Abilities (movement, combat, dragon abilities)
  • Implement Gameplay Effects (buffs, debuffs, cooldowns)
  • Design and implement skill tree UI and logic
  • Implement replication of abilities for co‑op

Dragon Companion System

+

Grow your baby dragon: AI behaviours, bonding, ability progression and co‑op sync.

  • Create dragon AI controller and behaviour tree (follow, hide, rest)
  • Implement bonding system based on player actions and proximity
  • Develop dragon ability progression integrated with GAS
  • Design dragon interaction UI (status, commands)
  • Set up replication and save/load for dragon state

Co‑op Networking

+

Enable multiplayer: sessions, replication and RPCs for all systems.

  • Configure Online Subsystem and session management
  • Set up actor/component replication for core systems
  • Implement Remote Procedure Calls (RPCs) for gameplay events
  • Test co‑op flow (joining, leaving, reconnection)

Quest & Mission System

+

Create story and side missions with mission manager and task objects; track progress and trigger events.

  • Design Mission Manager blueprint with an array of missions
  • Create base Mission class with start and completion events
  • Develop mission tasks and conditions
  • Integrate mission triggers into world (volumes, dialogues)
  • Implement mission replication and saving

AI & Enemies

+

Create AI for cultists and enemies with patrol, detection and combat behaviours.

  • Set up AI Controller and Behavior Tree for cultists
  • Implement perception components (sight, hearing)
  • Develop combat behaviours and animations
  • Integrate enemy abilities with GAS
  • Implement replication for enemy AI

Save/Load System

+

Persist data for inventory, missions, dragon state and world changes using SaveGame objects.

  • Create SaveGame blueprint and define storage variables
  • Implement logic to create or load SaveGame at game start
  • Write functions to read/write data to SaveGame
  • Integrate saving/loading with inventory, missions, dragon and world
  • Provide UI for save slots and player choice

UI & Interaction

+

Design and implement all interfaces: HUD, menus, inventory, dialogue, skill tree and dragon status.

  • Design HUD elements (health, stamina, bond meters)
  • Develop inventory UI and interaction panels
  • Create quest log and mission tracking UI
  • Implement dialogue interface with branching choices
  • Build skill tree and ability selection UI
  • Add dragon status and command interface

Open World & Streaming

+

Construct an open world environment and manage it with world partition, level streaming and performance optimisation.

  • Design world layout and identify major zones
  • Set up world partition or level streaming
  • Implement dynamic day/night and weather system
  • Add environmental interaction and destructible objects
  • Optimise LODs, culling and performance

Audio & VFX

+

Create immersive audio and visual effects: ambient sounds, voice overs, particle effects and dynamic music.

  • Compose or source ambient soundscapes
  • Implement sound cues and volumes for abilities and events
  • Create particle effects for fire, ash, magic and dragon abilities
  • Integrate dynamic music system that responds to gameplay
  • Optimise audio and VFX performance

Driveable Vehicles

+

Implement drivable vehicles such as cars or carts using Unreal’s vehicle systems and ensure co‑op replication.

  • Enable vehicle physics or Chaos Vehicles plugin
  • Create base vehicle class with physics, steering and throttle control
  • Implement driving input mappings (accelerate, brake, steering)
  • Design multiple vehicle types with different stats
  • Integrate vehicles into the open world (roads, spawn points)
  • Implement network replication for vehicle movement and state
  • Add vehicle damage and repair mechanics
  • Provide UI elements (speedometer, fuel gauge)

Weapon System

+

Develop a firearm system with guns, ammunition, attachments and combat mechanics.

  • Define base weapon classes (pistol, rifle, shotgun)
  • Implement shooting mechanics (hitscan/projectile, spread, recoil)
  • Add ammunition system and reload mechanics
  • Create attachments and weapon customisation
  • Integrate weapons into inventory and equipping system
  • Add visual and audio effects (muzzle flash, gunshot sounds)
  • Implement network replication for weapon state and firing events

Crafting & Shelter Construction

+

Allow players to gather resources, craft items and build shelters for survival.

  • Design resource types and harvesting mechanics
  • Implement crafting system with recipes and output items
  • Create crafting UI and feedback
  • Design buildable shelter structures with modular pieces
  • Implement placement system for structures
  • Add upgrade and repair mechanics for shelters
  • Integrate shelters and crafted items into Save/Load system
  • Tie advanced recipes and shelter upgrades into the skill tree
